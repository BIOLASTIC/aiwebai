# Skill: gateway-operator

## Overview
This skill teaches OpenClaw how to use the Gemini Unified Gateway tools.

## Instructions
- When asked to generate an image, use the `generate_image` tool.
- When asked for video, use `generate_video`.
- For music, use `generate_music`.
- For complex research, use `deep_research`.
- Always provide the user with the Job ID if a long-running task is started.
- Use the `chat` tool for standard conversational queries if standard models are not available.
